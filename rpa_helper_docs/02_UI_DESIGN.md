## 流程编辑器

```
--------------------------------------------------
| 流程：门诊查询                                  |
--------------------------------------------------
| 1. 点击图片：查询菜单     [测试] [删除]         |
| 2. 等待：1秒              [测试] [删除]         |
| 3. 输入文本：2026-05-23  [测试] [删除]         |
--------------------------------------------------
| [添加步骤] [保存流程]                           |
--------------------------------------------------
```

## 截图管理器

```
------------------------------------
| 截图管理                         |
------------------------------------
| 查询按钮      [重新截图]         |

```

## UX 原则

- 不暴露 YAML
- 不暴露代码
- 面向非技术用户
- 所有配置可视化
